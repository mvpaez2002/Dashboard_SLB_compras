## 📌 Descripción

Este proyecto consiste en un **dashboard interactivo de compras**, diseñado para analizar el comportamiento de adquisiciones y facilitar la toma de decisiones basada en datos.

El dashboard permite visualizar métricas clave relacionadas con compras, identificar patrones y detectar oportunidades de mejora en la gestión.

---

## 🎯 Objetivo

- Analizar datos de compras
- Identificar tendencias y comportamientos
- Facilitar la toma de decisiones estratégicas
- Visualizar información de forma clara e interactiva

---

## 📊 Funcionalidades

- Visualización de indicadores clave (KPIs)
- Filtros dinámicos para explorar los datos
- Análisis por categorías / proveedores / periodos
- Interfaz clara e intuitiva

---

## 🛠️ Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- Looker Studio

---

## 📈 Resultados / Insights

- Identificación de patrones de compra
- Detección de posibles ineficiencias
- Mejor comprensión del comportamiento de adquisiciones
